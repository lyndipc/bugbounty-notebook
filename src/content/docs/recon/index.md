title: "Reconnaissance"
---

Recon determines the size and shape of the attack surface. Better recon = more unique bugs that other hunters miss. These pages cover each recon phase with copy-paste commands.

Replace placeholders:
- `<TARGET>` — root domain (e.g. `example.com`)
- `<SUBDOMAIN>` — specific subdomain
- `<IP>` — target IP address
- `<WORDLIST>` — path to wordlist
