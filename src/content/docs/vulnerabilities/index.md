title: "Vulnerabilities"
---

Deep dives into each vulnerability class relevant to bug bounty hunting. Each page covers detection, exploitation, bypasses, and reporting considerations.

For each bug class, the structure is:
- **What it is** — quick refresher
- **Where to look** — which features / endpoints are typically vulnerable
- **Testing** — manual steps and payloads
- **Bypasses** — when basic payloads get filtered
- **Impact** — how to demonstrate real-world impact for your report
- **Findings** — space to log what you found on this target
