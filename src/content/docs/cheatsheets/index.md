title: "Cheatsheets"
---

Copy-paste-ready command references. During testing, grab the command and adapt it — no thinking required.

Replace placeholders:
- `<TARGET>` — root domain (e.g. `example.com`)
- `<SUBDOMAIN>` — specific subdomain
- `<URL>` — full URL including scheme
- `<CALLBACK>` — your callback server (Burp Collaborator, interactsh, etc.)
- `<COOKIE>` — session cookie value
- `<WORDLIST>` — path to wordlist
